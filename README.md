# laravel-example

Установка
=========

Самый простой способ развернуть проект - воспользоваться docker-compose. 
Для этого заходим в папку docker и вызваем команды make

```
make prepare-project
make create-project
```

После этого заходим на сайт по http://localhost:8081. 
Если хотим посмотреть на базу, то заходим на http://localhost:5433.
Доступы взять из .env файла, чтобы создать подключение.



Без докера - придется руками создавать хост в своем веб-сервере. Разворачивать базу. Копировать .env.example в .env, прописав свои доступы к БД. И применить миграции доктрины, командой 

```
php artisan doctrine:migrations:migrate
```
